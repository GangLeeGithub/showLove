一个向女神表白的App

前言：

我向来不善言辞，不管是说出来也好还是写在纸上也罢。但是表白这件事绝不能含糊，我想力所能及的让这份情书看起来更靚眼一些。恰巧我看到了一个表白的网页，感觉很棒，但是有不少硬伤：手机竖屏体验非常不好，缺少必要交互，缺少上报机制--“对方看没看完，想法如何”都不知道。由于我不擅长网页制作，于是在我相对擅长的Android端做了这个应用，希望可以打动对方。
